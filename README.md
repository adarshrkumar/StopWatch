# StopWatch

A precise online stopwatch with millisecond accuracy.

## Features

- **Millisecond Precision**: Displays time as HH:MM:SS:MS
- **Seconds Only Mode**: Toggle to show total elapsed seconds with decimals
- **State Persistence**: Saves time to localStorage
- **Mobile Friendly**: Responsive design with PWA support

## Controls

- **Start/Stop**: Begin or pause the timer
- **Reset**: Clear the timer back to zero
- **Seconds Only Toggle**: Switch between normal and seconds-only display

## Display Modes

### Normal Mode
```
00:05:23:45
(hours:minutes:seconds:milliseconds)
```

### Seconds Only Mode
```
323.45s
(total seconds with 2 decimal places)
```

## Tech Stack

- HTML/CSS/JavaScript
- LocalStorage for persistence
- Hosted via GitHub Pages
